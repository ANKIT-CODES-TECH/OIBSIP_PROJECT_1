# Online Reservation System

## Project Description

This project is a simple Java-based Online Reservation System.

It allows users to:

- Login using valid credentials
- Reserve train tickets
- Generate a PNR number
- Cancel tickets using PNR number

## Modules

### 1. Login Form

Users must enter a valid Login ID and Password to access the system.

### 2. Reservation Form

Users enter:

- Passenger Name
- Train Number
- Class Type
- Date of Journey
- From Place
- Destination

The train name is automatically displayed based on the train number.

### 3. Cancellation Form

Users enter a PNR number to view reservation details and cancel the ticket.
